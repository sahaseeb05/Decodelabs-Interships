# Recommendation Engine (RecEngine) 🚀

This repository contains a Machine Learning-based Recommendation Engine built with Python and Flask. It demonstrates text vectorization and similarity matching to provide accurate recommendations based on user input or item descriptions.

---

## 🛠️ Tech Stack & Tools

* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3
* **Machine Learning:** Scikit-Learn (TF-IDF Vectorizer / Cosine Similarity)
* **Deployment & Version Control:** Git, GitHub
